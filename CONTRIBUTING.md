# Contributing to BitYield

BitYield is currently in the validation stage. The immediate work is user
research, prototype design, and market testing — not engineering.

Contributions welcome right now:
- User research: help recruit non-technical Bitcoin holders for interviews
- Design: Figma mockups for the deposit flow and dashboard
- Documentation: improve any page in the docs/ folder
- Research: identify Bitcoin holder communities worth testing with

Contributions that open after validation is complete:
- Clarity smart contract development (YieldRouter, PositionNFT)
- Next.js frontend implementation
- Stacks.js and Leather Wallet integration
- Clarinet test suites for all contracts

If you want to contribute now, open an issue describing what you want
to help with. If you want to be notified when engineering contributions
open, watch this repository.

Code of conduct: be specific, be constructive, assume good intent.
